# GitHub Member Directory

GitHub Member Directory is a web app that pulls and displays GitHub user information in a member card.

## Installation

To run the web app locally:

1. Clone the repo
2. Install vite
    ```bash
    yarn add vite
    ```
3. Run app
    ```bash
    yarn dev
    ```
